Blind Tiger
August 9th, 2020

Simple Password Manager

I wanted to learn more about cryptography and software development so I decided to make a password manager. It's for personal use and as a portfolio piece, however, if others want to use it I can't stop them.

I do not recommend using Blind Tiger as I am not a cyber security professional. It's primarily a learning project where I can apply new knowledge of cryptography and software development.

Based on the algorithm and ideas presented in "Computers & Graphics, ISSN: 0097-8493, Vol: 37, Issue: 5, Page: 333-347"
