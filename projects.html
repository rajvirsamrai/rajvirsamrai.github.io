Blind Tiger

In Progress

An Amateur Text Encryption Tool

Use Blind Tiger to encrypt and compress plain text quickly. You can choose from the available compression and encryption methods as wells as the order in which they're done.

This project started because I wanted to learn about compression, cryptography, and software design patterns. I also wanted to confirm to myself I could program in Java.

You can also interact with Blind Tiger through the shell.

  • Source Code via GitHub
  • Made with JavaFX
